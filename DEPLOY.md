# 宝宝如厕训练助手 · 云端测试部署流程

---

## 总览

本地编译成功只是第一步，要让他人在自己微信里扫码体验，需要经过以下流程：

```
本地编译 → 上传代码 → 设为体验版 → 添加体验者 → 扫码测试 → 提交审核 → 发布上线
```

---

## 第一步：注册真实小程序 AppID

当前 `project.config.json` 里的 `appid` 是 `touristappid`，只能本地调试。要部署云端测试，必须更换为真实 AppID。

1. 打开 **[微信公众平台](https://mp.weixin.qq.com/)**，注册小程序的 AppID（个人/企业均可）
2. 登录后进入「开发」→「开发管理」→「开发设置」，复制你的 **AppID**

> 个人主体注册即可，免费，不需要认证费。功能限制：不能使用微信支付和部分高级 API，但你这个如厕训练助手完全够用。

---

## 第二步：配置 AppID

编辑 `project.config.json`，将 `appid` 改为你自己的真实 AppID：

```json
"appid": "wx1234567890abcdef"
```

> **注意**：改完后重新在微信开发者工具中打开项目，工具会自动识别新的 AppID。

---

## 第三步：配置域名白名单（如需要）

你的小程序目前使用**本地存储**（`USE_CLOUD = false`），不需要网络请求，所以**这一步可以跳过**。

如果将来接入云开发（`USE_CLOUD = true`），需要在「开发管理」→「开发设置」→「服务器域名」中配置：
- request 合法域名：你的 API 域名
- uploadFile 合法域名：头像上传域名（云开发模式下可自动跳过）

---

## 第四步：上传代码

在**微信开发者工具**中：

1. 确保项目编译通过，功能正常
2. 点击右上角 **「上传」** 按钮
3. 填写版本号（如 `1.0.0`）和版本描述（如「首次上传，修复添加记录人 bug」）
4. 点击上传

上传成功后，代码不会立刻生效，需要到后台管理。

![上传按钮在开发者工具右上角](tool-top-right-upload.png)

---

## 第五步：设为体验版

在 **[微信公众平台](https://mp.weixin.qq.com/)** 中：

1. 左侧菜单 → **「管理」** → **「版本管理」**
2. 在「开发版本」区域找到刚上传的版本
3. 点击右侧 **「选为体验版」**
4. 确认后，会生成一个**体验版二维码**

> 每次上传新代码后都需要重新设为体验版。

---

## 第六步：添加体验者

1. 左侧菜单 → **「管理」** → **「成员管理」**
2. 切换到 **「体验成员」** 标签
3. 点击「添加成员」，输入对方的**微信号**（不是昵称，是注册时的微信号）
4. 添加后，对方微信会收到通知

> 体验者上限：个人主体 15 人，企业主体 60 人。

---

## 第七步：扫码测试

1. 将第五步生成的**体验版二维码**发给体验者
2. 体验者用微信扫码 → 点击「前往体验版」
3. 开始测试各项功能

**测试要点**：
- [ ] 首次打开能看到锁屏页，输入密码 `0000`（config.js 中 LOCK_PASSWORD 的默认值）能进入
- [ ] **添加记录人**：点击「当前记录人」→「添加照顾者」→ 选择头像 → 输入昵称 → 点「保存」，能成功添加并自动选中
- [ ] 快速记录（小便/大便/换小内裤/换尿不湿）正常工作
- [ ] 分析页图表正常显示
- [ ] 历史页记录可展开、删除
- [ ] 撤销功能正常

---

## 第八步：提交审核（可选，准备对外发布时做）

1. 版本管理 → 开发版本 → 点击 **「提交审核」**
2. 填写审核信息：
   - 功能页面：pages/record/record
   - 测试账号：无（你的小程序不需要登录）
   - 类目：工具 → 信息查询
3. 等待 1-7 个工作日审核
4. 审核通过后点击「发布」，小程序就全量上线了

---

## 常见问题

### Q: 上传时报错「未找到 AppID」
A: `project.config.json` 中的 `appid` 还是 `touristappid`，改为真实 AppID 后重新打开项目。

### Q: 体验者扫码后打不开
A: 检查：
1. 体验者是否在「体验成员」列表中
2. 体验版是否已设为最新上传的版本
3. 对方微信版本是否过低

### Q: 改密码在哪改？
A: 编辑 `config.js` 中的 `LOCK_PASSWORD`，重新上传即可。密码是 4 位数字字符串。

### Q: 想多人共享数据怎么搞？
A: 按 `config.js` 中的注释开启云开发：
1. 微信开发者工具 → 顶部「云开发」→ 开通免费版
2. 创建集合 `potty_records`
3. `config.js` 中设置 `USE_CLOUD = true`，填写 `ENV` 为你的环境 ID
