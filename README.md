# 宝宝如厕训练助手 · 微信小程序

基于已规划 UI 实现的**原生微信小程序**（WXML / WXSS / JS，无需 npm 构建）。

## 功能
- **启动锁屏**：微信授权 → 输入 4 位固定密码（仅首次），正确后进入，之后不再弹锁屏；错误则红点抖动提示
- **记录**：快速记录小便 / 大便 / 换小内裤 / 换尿不湿，含今日统计、预测面板、撤销
- **分析**：排便间隔趋势、常见排便时段、近 7 天堆叠图
- **历史**：按日分组、可展开、单条删除、清空全部

## 运行
1. 微信开发者工具 → **导入项目** → 选择本目录
2. **AppID**：本地预览用 `touristappid`；正式发布换成你自己的 AppID（改 `project.config.json`）
3. 打开 `config.js`，把 `LOCK_PASSWORD` 改成你想要的 4 位密码
4. 编译运行

> 重测锁屏：开发者工具 → 清缓存 → 全部清除（清掉 `app_verified`），下次启动重新弹锁屏。

## 目录结构
- `pages/lock` —— 启动锁屏（授权 + 一次性密码）
- `pages/record` `pages/analysis` `pages/history` —— 三个 Tab
- `utils/storage.js` —— 记录本地存储
- `utils/analysis.js` —— 预测 / 趋势 / 日均统计
- `config.js` —— 配置（密码、存储 key、类型语义色）
